# MRG Research Competition Website

Competition microsite scaffold for MYResearchGuide. It includes:

- competition timeline
- FAQ
- researcher panel with external and UTAR categories
- available projects and application directories
- sponsor, partner, and UTAR organisation sections
- main-site links back to [MYResearchGuide](https://www.myresearchguide.org/)

## Development

```bash
npm install
npm run dev
```

## Verification

```bash
npm run typecheck
npm run lint
npm run build
```

The project is a Vite React app using Tailwind CSS v4 and shadcn/ui source components.
